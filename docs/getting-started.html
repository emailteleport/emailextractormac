Getting Started

This guide will help you download, install, and run your first email extraction with EmailExtractorMax.

Step 1: Download

Download EmailExtractorMax from our website:

  1. Go to the download page
  2. Click the "Download for macOS" button
  3. The DMG file will start downloading

Step 2: Install

Install EmailExtractorMax on your Mac:

  1. Open the downloaded .dmg file
  2. Drag EmailExtractorMax to your Applications folder
  3. Eject the DMG and you're done!

First Launch

When you first open EmailExtractorMax, macOS may display a security warning. To proceed:

  1. Right-click (or Control-click) on the app
  2. Select "Open" from the menu
  3. Click "Open" in the dialog that appears

You only need to do this once. Future launches will work normally.

Step 3: Your First Extraction

Let's run a simple email extraction using search engines:

1. Select Search Engine Mode

In the main window, click on the "Search Engines" tab. This is the most common way to find new email addresses.

2. Enter Your Search Query

Type a search query in the search box. For example: 

"marketing manager" "email" site:linkedin.com

This search will look for marketing managers on LinkedIn whose email addresses might be publicly listed.

3. Select Search Engines

Check the search engines you want to use. For best results, enable multiple engines to search in parallel.

4. Start the Search

Click the "Start" button. EmailExtractorMax will:

  • Query the selected search engines
  • Visit each result page
  • Extract any email addresses found
  • Display results in real-time

5. Review Results

As emails are found, they'll appear in the results table. You can see:

  • Email address
  • Source URL where it was found
  • Search engine that found it
  • Page title for context

Step 4: Export Your Results

Once you're satisfied with your results:

  1. Click the "Export" button
  2. Choose your format (TXT or CSV)
  3. Select where to save the file
  4. Click "Save"

Next Steps

Tips for Better Results

  • Use specific queries - More targeted searches yield more relevant results
  • Enable multiple engines - Different engines return different results
  • Set up filters - Use blacklists to exclude unwanted domains
  • Try deep crawling - Enable this to find more emails on linked pages
